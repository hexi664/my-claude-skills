## 核心定位

> **Widget Together, Share Whatever!**

Widgetable 是一款**社交小组件 App**，让用户通过 iOS/Android 锁屏和主屏小组件与闺蜜、情侣保持互动连接。核心价值是**把手机屏幕变成情感连接的入口**——不是聊天工具，而是用可爱、轻量的方式让亲密关系"可见"。

## 目标用户画像

### 核心用户
- **年龄**：13-25 岁（Z 世代为主）
- **性别**：女性偏多
- **关系**：有闺蜜/情侣关系，特别是异地恋 (LDR)
- **特征**：喜欢个性化、可爱风格、社交分享
- **地区**：全球，东南亚/东亚/北美为主

### 使用动机
1. **情感连接**：和远距离的朋友/伴侣保持"在场感"
2. **个性化表达**：让手机屏幕体现自己的审美和关系
3. **轻量互动**：不想打字聊天，用小组件传递心意
4. **养成乐趣**：虚拟宠物/植物的陪伴感和成就感

## 核心功能模块

### 1. 社交互动组件
| 功能 | 描述 | 数据来源 |
|------|------|----------|
| Distance Widget | 实时显示与好友/伴侣的物理距离 | GPS 定位（后台） |
| Miss You Widget | 点击发送"想你"，计数器累加 | 服务端同步 |
| Status Widget | 显示好友当前状态（睡觉/吃饭/运动等） | 用户手动更新 |
| Pin It! | 发送照片/表情/涂鸦/文字到对方屏幕 | 服务端推送 |
| Mood Bubble / Mood Jar | 表达和分享情绪 | 用户输入 |
| Sleep Widget | 追踪并分享彼此睡眠状态 | 用户记录 |
| Friends Note | 在对方锁屏留言 | 服务端推送 |
| Little Signal | 发送趣味信号（炸弹/亲吻等） | 服务端推送 |
| Step Counter | 与好友设定每日步数目标并互相追踪 | HealthKit/本地 |

### 2. 虚拟养成系统
| 功能 | 描述 |
|------|------|
| Virtual Pet | 领养虚拟宠物，喂食/玩耍/成长 |
| Co-parenting | 和好友/伴侣共同养宠物 |
| Pet Egg / Pet Food | 虚拟货币购买宠物蛋和食物 |
| Plant Widget | 在屏幕上种植虚拟植物 |

### 3. 个性化定制
| 功能 | 描述 |
|------|------|
| Trending Wallpapers | 潮流壁纸（3D/AI/剪纸风等） |
| Dynamic Icons | 将静态图标变为动画图标 |
| Theme Matching | 和好友匹配主题风格 |
| Background/Font/Border | 小组件外观自定义 |
| App Shortcut Widget | 锁屏快捷打开常用 App |

## 竞品对比

| 竞品 | 定位 | 差异 |
|------|------|------|
| **Locket Widget** | 情侣/朋友照片小组件 | 专注照片分享，功能更简单 |
| **BondBee** | 情侣日常互动 | 情侣专属，无闺蜜场景 |
| **Widgetsmith** | 个人小组件定制 | 无社交互动，纯个人美化 |
| **WidgetClub** | 主屏/锁屏美化 | 偏主题和壁纸，无养成系统 |
| **Bear Countdown** | 倒计时小组件 | 单一功能 |
| **Between** | 情侣专属空间 | 重聊天和相册，非小组件形态 |

**Widgetable 的独特优势**：
1. **社交 + 养成 + 美化** 三位一体（竞品通常只做其一）
2. 同时覆盖**情侣 + 闺蜜**两大关系场景
3. 小组件作为核心载体（非聊天/信息流）
4. 虚拟宠物共养创造强粘性

## 设计风格特征

- **整体调性**：可爱 (Cute)、活泼 (Playful)、年轻 (Youthful)
- **色彩**：高饱和度暖色系为主，粉/紫/橙/黄
- **插画风格**：圆润卡通、Kawaii 风
- **动效**：弹跳、闪烁、心跳等情感化动效
- **排版**：圆角卡片、大圆角按钮、气泡对话框
- **图标**：填充风格、圆润线条

## 技术特征（外部可观察）

- **iOS WidgetKit**：锁屏+主屏小组件
- **后台定位**：Distance Widget 需要持续定位
- **实时数据同步**：Pin It / Miss You / Status 等需要服务端推送
- **HealthKit 集成**：步数追踪
- **数据隐私**：收集位置、健康数据、使用数据（App Store 标注）
- **年龄限制**：13+（含卡通暴力、粗俗幽默、战利品箱）

## 用户痛点（来自评论分析）

### 正面反馈
- 创意有趣，和朋友互动方式新颖
- 个性化选项丰富
- 虚拟宠物很可爱有粘性

### 负面反馈
- 广告过多（免费用户）
- 部分高级功能锁在付费墙后
- 偶发 Bug 和卡顿
- 宠物养成需要内购才能顺畅
- 部分语言翻译质量差
- 导航有时不够直观
